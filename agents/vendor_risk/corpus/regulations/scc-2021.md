# EU Standard Contractual Clauses (2021 module) — summary

## Modules
- **Module 1**: Controller → Controller
- **Module 2**: Controller → Processor (most common for SaaS DPA)
- **Module 3**: Processor → Processor
- **Module 4**: Processor → Controller

## Key Differences vs 2010 SCCs
- 2021 SCCs incorporate Schrems II requirements (transfer impact assessment).
- 2010 SCCs were SUPERSEDED and are no longer valid for new contracts after
  December 27, 2022.

## TPRM Verification
- A DPA referencing the 2010 SCC module is non-compliant for new
  arrangements. Flag as a HIGH conflict.
- DPA must include a Transfer Impact Assessment (TIA) clause.
