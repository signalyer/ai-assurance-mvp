Assurance Model Providers
AI supply-chain governance for the models the platform uses to evaluate, judge, summarize, and explain — separate from production runtime models.
Assurance ≠ production inference. These providers are governed for evaluation, summarization, and governance Q&A. Restricted data (PII / NPI / PCI / payment / AML-KYC / credit) requires AWS Bedrock or Local/VPC unless an exception is approved. All model-assisted analysis is logged.
